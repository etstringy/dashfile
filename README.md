### **Dashfile**

Create files from Geometry Dash levels, and add them to your game with this simple program.

## [Download here!](https://github.com/etstringy/dashfile/releases)

Features
- Create .dash files
- Add .dash files to your GD save file
- Dark/Light themes
- Auto-updates!

Development:
```
yarn
yarn start
yarn build
```
